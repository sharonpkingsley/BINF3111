Getting Started

Please ensure all databases are uploaded before search.

1.Upload Page


1.1Update database


1.1.1 File format

The evidence file should be in csv format. The content should be in the form of a p*p matrix where entry(i,j) is the similarity score between drug i and drug j. Drugbank IDs are used to represent drugs in title columns and rows. 'DRUGBANK_ID' should be labelled in the entry(0,0).

The info file should be in csv format. The info file contains a reference sheet for DRUGBANK_ID and GENERIC_NAME. There should be two columns in the file. The first column lists DRUGBANK_IDs, whereas the second column lists the corresponding GENERIC_NAMEs.


1.1.2 File type

Click on the dropdown list and choose the database you want to update. target, pathway and chemical_structure are three different evidence types. info is the file which gives the generic names according to the drugbank_ids which drugs are represented as in the evidence files.


1.2 Add a new drug


1.2.1 New drug name and ID

Enter new drug ID in the 'New Drug ID' textbox. Example: DB30000. Enter new drug name in the 'New Drug Name' textbox. Example: Goserelin. Both textboxes have autosuggestion that reminds you what IDs/names already exist in the database. Please make sure no duplicates are added to the database.


1.2.2 New drug file

The new drug file should consist of two columns: drugbank_ID and score. The first column should have all drugbank_IDs in the database plus itself. The second colomn should have the similarity score to the corresponding drug.


1.2.3 Evidence type

Choose the evidence type the score is in the dropdown list.


1.3 Edit a drug record


1.3.1 Drug A/B name

Enter the name/ID of the drugs of which the similarity score you want to change in Drug A/Drug B name textboxes. Both textboxes have autocomplete function. Choose one of the 'ID Name' from the dropdown list.


1.3.2 Score

Enter the new similarity score into the textbox. Only score from 0-1 is allowed. You can use the (?adjuster) on the right to change score in a step of 0.01.


1.3.3 Evidence type

Choose the evidence type the similarity score is in the dropdown list.



2. Search Page


2.1 Drug Name

Enter the name/ID of the drug of which the similarity score you want to search in Drug name textbox. Choose one of the 'ID Name' from the autocomplete dropdown list.


2.2 Filter Options


Click on Filter Options to discose the options available.

2.2.1 Evidence type

Tick the checkbox of evidence(s) which you want to search within. Tick 'All' to select all evidences at once.


2.2.2 Threshold

Enter a score between 0-1 in the threshold textbox. Only the drugs with a similarity higher than the threshold will be displayed in the result.


3. Result Page


3.1 Network view


3.2 Table view

The table consists of at least four columns: ID, Name, score_of_an_evidence(up to columns), Score. The last column score is an average of all existing score_of_an_evidnece.


3.3 Export

The table view can be exported as a csv file. Click on the 'Download' button and the file will be downloaded to the default location.


4. Others


4.1 Index Page

Click on search leads to search page.
Click on upload leads to upload page.

4.2 Buttons on top

Home button leads back to index page.
Back button leads back the previous page.
Help button leads to the help page.
